# Osm-Api

## Installation

```
$ git clone https://github.com/Position-Platform/osm-api.git
$ cd osm-api
$ cp .env.example .env
$ docker compose up -d
```

- import .osm.pbf file in osm folder
- Exple folder : /src/osm/cameroun.osm.pbf

```
$ docker exec -it osm_api bash
$ cp .env.example .env
$ npm run exec
```
